## typescript

### typescript란?

* JS를 기본으로 한 JS 의 superset.
* statically typed : 코딩할때 타입 결정 => 컴파일 도중에 에러를 잡을 수 있다. 
* 객체지향형 프로그래밍에 더 적합. // class, interface 등
